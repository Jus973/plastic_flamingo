Flamboyance Playground

A deliberately designed multi-page fixture site for testing Flamboyance, a tool that launches Playwright-based synthetic users against your app. Each scenario below is crafted to trigger specific frustration signals or test persona behaviors.

Main Flows

  • Shop Start of the purchase flow - browse products
  • Cart Mid purchase flow - review cart items
  • Checkout End state for purchase with success message
  • Account Account area index page
  • Account Settings Notifications, password, preferences
  • Sign Up Simple signup form with client-side confirmation
  • Order Status Order lookup by order number

Stress Test Scenarios

  • Circular Navigation A ↔ B loop designed to trigger circular_navigation signal
  • Rage Decoy Non-interactive elements that look clickable
  • Hidden Menu Collapsed nav (aria-expanded) vs visible escape hatch
  • Icon Bar Icon-only actions vs labeled alternatives
  • Many Links 35+ interactive targets to stress sampling behavior
  • Slow Page Delayed content for testing low-patience personas
