You've Reached a Dead End

This page intentionally has no clickable elements. There are no links, no buttons, and no interactive elements whatsoever.

This scenario is designed to trigger the dead_end frustration signal when Flamboyance detects that there are no clickable elements on the page.

Dead End Detection

The record_dead_end(url) function emits a dead_end signal when no clickable elements (links, buttons, or other interactive elements) are found on the page.

Users encountering this page have no way to navigate forward through the UI. They must use browser navigation (back button) or type a new URL manually.

Expected behavior: Synthetic users should identify this as a frustration point and either navigate back or report the dead end.

How to Escape

Since there are no clickable elements, you must use your browser's back button, keyboard shortcut (Alt+Left or Cmd+[), or manually navigate to another URL.
